Sustainability

Long-term impact, preservation, and future directions

Project Longevity

This project is designed to function as a long-term digital resource that can be accessed beyond the classroom. By hosting the site on GitHub Pages and using widely supported web technologies (HTML, CSS, and ArcGIS StoryMaps), the project remains stable, accessible, and easily updatable.

Digital Preservation

Sustainability in digital history requires attention to preservation and accessibility. This project prioritizes:

  • Use of open and non-proprietary formats (HTML, CSS)
  • Reliable hosting through GitHub Pages
  • Integration with established platforms such as ArcGIS
  • Clear organization of content for future updates

These choices ensure that the project can be maintained, expanded, and reused over time.

Public History & Accessibility

This project is designed not only for academic audiences but also for the general public. By presenting historical content through an interactive and visually engaging format, it makes complex histories more accessible.

The use of narrative structure, mapping, and visual materials encourages users to engage with history in a more immersive and meaningful way.

Future Development

  • Expanding the archive with additional primary sources
  • Adding more Indigenous perspectives and voices
  • Enhancing the interactive map with additional data layers
  • Incorporating oral histories or multimedia content
  • Developing educational materials for classroom use

These expansions would deepen the project’s interpretive scope and broaden its impact.

Reflection

The ongoing relationship between Ireland and Indigenous nations demonstrates that history is not static—it continues to shape present-day connections and acts of solidarity. This project reflects that dynamic by presenting history as an evolving conversation rather than a fixed narrative.

By combining digital tools with historical research, this project contributes to a broader effort to make history more accessible, inclusive, and responsive to contemporary audiences.

Rachel Pasternack | Digital History Project | Spring 2026
