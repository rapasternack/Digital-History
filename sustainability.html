Platform and Structure

This site is a static HTML site hosted on GitHub Pages. It uses no content management system, no server-side processing, and no external JavaScript frameworks. All pages are plain HTML files linked to a single stylesheet. This structure was chosen because it is transparent, lightweight, and does not depend on any third-party platform remaining functional.

Static sites are well suited to preservation. The files are human-readable, browser-renderable without any build step, and do not depend on a database or a running application. The same files that are in the repository are the files that produce the site.

Essential Files

The following files are required for the site to function:

  • index.html — home page
  • planning.html — interface sketches and design documentation
  • findings.html — central artifact and interpretation
  • sustainability.html — this page
  • style.css — all visual styling
  • assets/holc-map.png — the map artifact displayed on the findings page
  • assets/sketch-home.jpg, sketch-findings.jpg, sketch-sustainability.jpg — interface sketch photos displayed on the planning page

If the assets/ folder or any of its contents are missing, images will fail to load but the text of the site will remain intact.

Data and Dependencies

The map on the findings page was produced from two external datasets:

  • HOLC boundary data: Mapping Inequality, University of Richmond Digital Scholarship Lab. Downloaded as a GeoJSON file in February 2026. The data is openly licensed and redistributable.
  • Census income data: U.S. Census Bureau American Community Survey 5-Year Estimates (2020), retrieved via the Census API. The retrieval script is in the project repository.

The Python script used to produce the map is in the repository at scripts/make-holc-map.py. The script requires GeoPandas, Matplotlib, and the Census API key stored as an environment variable. The map is static — it does not call any live API when the page loads.

There are no other external dependencies. The CSS is self-contained. No CDN resources are loaded.

Maintaining and Extending the Site

To update or extend this site after the course ends, a future user would need:

  1. A GitHub account with access to the repository
  2. Basic familiarity with HTML and CSS to edit page content
  3. Python with GeoPandas and Matplotlib to reproduce or update the map
  4. A Census API key to re-run the data retrieval script

No other tools, accounts, or software are required. The site can be cloned, edited locally, and pushed back to GitHub Pages without any build process. If GitHub Pages becomes unavailable, the files can be hosted on any static file server without modification.
