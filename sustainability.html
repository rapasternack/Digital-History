Sustainability

Technical documentation, platform choices, and project maintenance

Platform Choices

  • HTML5 – Provides the structural framework for all pages
  • CSS3 – Controls layout, typography, and visual design
  • GitHub Pages – Static hosting for long-term public access
  • ArcGIS Online – Used for interactive mapping and spatial visualization
  • Google Fonts – Ensures consistent typography across devices

Essential Project Files

The website is structured using a small set of core files. These files must remain organized together for the project to function correctly.

  • index.html – Homepage introducing the project
  • findings.html – Timeline, map, and archival content
  • planning.html – Research design and development process
  • sustainability.html – Technical documentation and maintenance notes
  • style.css – Styling for all pages
  • assets/ – Folder for images and media

What a Future User Would Need

A future maintainer or researcher should have access to the following in order to update or preserve this project:

  • Basic understanding of HTML and CSS
  • Access to the GitHub repository hosting the project
  • ArcGIS account (only if updating or replacing the interactive map)
  • Image source URLs or local image files if external links break
  • Awareness of page structure so links between sections remain intact

Maintenance Notes

  • Ensure all external image links remain active or are replaced with local files if necessary
  • Check ArcGIS embeds periodically for access or permission issues
  • Update bibliography and archive links if sources move or change
  • Maintain consistent navigation structure across all pages
