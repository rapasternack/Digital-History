Aims of Database

The database has not been created yet but aims to include documents such as letters, newspaper articles, and oral histories related to Irish–Indigenous interactions. It will be designed to be searchable and filterable by date, location, and type of source.

Primary Source Database

This database compiles primary sources related to Irish–Indigenous interactions, including letters, newspaper articles, and oral histories. It is designed to be searchable and filterable by date, location, and type of source.
